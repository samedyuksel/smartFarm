# Smart Greenhouse Automation
This project was developed by Samed YÃ¼ksel as open source for a private company on 2019. This project include nextion screen for HMI. With this, you can instantly learn the temperature and humidity of the environment and soils. You can adjust what should be done (such as irrigation) according to these values.

## Materials Required
Arduino Mega
* Nextion Screen
* Wemos d1 mini (optional for mobile control)
* voltage regulator
* DS3231 RTC Module
* 10K NTC
* Solenoid Valve
* Soil moisture sensor
* pH Sensor
* 8 Channel Relay Board
